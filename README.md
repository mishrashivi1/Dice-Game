🎲 Dice Game (Vite + React)

A simple and interactive dice game built using React (Vite) where users can select a number and roll the dice to earn points.

🚀 Features

🎯 Select a number between 1–6
🎲 Click the dice to roll
📈 Score increases if your guess is correct
📉 2 points deducted for wrong guess
🔄 Reset score anytime
📜 Toggle game rules
🛠️ Tech Stack
⚛️ React (Vite)
🎨 CSS
🧠 JavaScript (Logic handling)

🎮 How to Play
Select any number (1–6)
Click on the dice to roll
If your selected number matches the dice:
You gain points equal to the dice number
If wrong:
2 points will be deducted
⚙️ Installation & Setup

Clone the repo:

git clone https://github.com/mishrashivi1/Dice-Game

Go into the project folder:

cd Dice-Game

Install dependencies:

npm install

Run the project:

npm run dev
